# Knwl API

[Knwl](https://github.com/Orbifold/knwl) is a flexible Graph RAG package with advanced ingestion and augmentation strategies. 

This repository provides two interfaces for Knwl:
- **FastAPI REST API**: HTTP/REST interface for web applications
- **MCP Server**: Model Context Protocol interface for AI assistants and agents

## Quick Start

### FastAPI Server

Run the FastAPI web server:

```bash
uv run uvicorn knwl_api.main:app --reload
```
